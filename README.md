# Useful Hardhat Commands

```shell
npm run <command> [-- <args>]
```

To run tests:
```shell
npm run test -- --network {network}
```

To deploy the contracts:
```shell
npm run deploy --network {network}
```

To build the source code:
```shell
npm run build
```
